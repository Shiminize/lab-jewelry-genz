Your filter pills are stacking because the parent container allows wrapping. Here's the exact fix:

‚ùå Current Problem Code:

  <div className="relative flex-shrink-0">
    <FilterPill label="Category" />
  </div>
  <div className="relative flex-shrink-0">
    <FilterPill label="Price" />
  </div>
  {/* ...more pills */}
</div>

<!-- ISSUE: The flex container might still wrap if parent doesn't constrain width -->

‚úÖ Fixed Code (Option 1 - Recommended):

<!-- Outer wrapper controls overflow -->
<div className="overflow-x-auto scrollbar-hide pb-2">
  <!-- Inner flex must not wrap, uses min-w-max -->
  <div className="flex items-center gap-3 min-w-max">
    <div className="relative flex-shrink-0">
      <FilterPill label="Category" />
      {isCategoryOpen && <FilterDropdown />}
    </div>
    <div className="relative flex-shrink-0">
      <FilterPill label="Price" />
      {isPriceOpen && <FilterDropdown />}
    </div>
    <div className="relative flex-shrink-0">
      <FilterPill label="Metal" />
      {isMetalOpen && <FilterDropdown />}
    </div>
    {/* ...more pills */}
  </div>
</div>

<!-- KEY CHANGES:
  1. Remove flex-nowrap from outer div (doesn't work with overflow-x-auto)
  2. Add min-w-max to inner flex container (prevents wrapping)
  3. Keep flex-shrink-0 on each pill wrapper
-->

üìè CSS Key: min-w-max

Forces the inner flex container to be as wide as its content, preventing wrapping even when parent has overflow.

üéØ Structure: Two Divs

Outer div handles scrolling, inner div holds flex items. This separation is crucial for preventing wraps.

üîí Each Pill: flex-shrink-0

Prevents individual pills from shrinking below their content size, maintaining consistent sizing.
