Mineral Verdant

Mineral Verdant mirrors Shihara Lab’s charcoal interiors with a seasonal jade accent. This demo visualizes the token replacements so Tailwind-driven components inherit the refresh with minimal override risk.

Palette Anchors

--color-bg #FCFDF7 · Page background
--color-surface #FFFFFF · Surface panels
--color-ink #111111 · Headline text
--color-ink-2 #405346 · Body text
--accent-secondary #6BBD4F · Accent highlight
--accent-muted #C7F79A · Soft badge/background

Type & Copy Sample

“Mineral depth, verdant lift.”

The primary button stays anchored in evergreen charcoal while secondary CTAs pick up seasonal jade for energy. Muted green-grays keep typography relaxed yet accessible across light surfaces.

Featured Piece

Verdant Column Necklace

Usage of #6BBD4F is limited to highlights and callouts, retaining the brand’s minimalism while bringing freshness to seasonal drops.

Token Mapping

Design Token Value Tailwind Utility Impact
